Algoritmo de Divisăo Inteira:

Problema: dados 'a' e 'd' inteiros, determinar 'q' e 'r', tais que podemos escrever

	a = q.d + r,	sendo q e r inteiros, e 0 <= r < |d|.	(1)

Neste contexto, 'a' é chamado o dividendo, 'd' o divisor, 'q' é o quociente, e 'r' é o resto. O Teorema da Divisăo Inteira assegura uma soluçăo única, e apresentamos aqui um breve algoritmo para encontrarmos esta soluçăo em uma situaçăo em que as únicas operaçőes aritméticas disponíveis săo soma e subtraçăo.

Quando 'a' e 'd' săo ambos năo negativos, a soluçăo é intuitivamente fácil. Basta subtrairmos 'd' de 'a' recursivamente, até obtermos um resultado menor que 'd'. O número de vezes que 'd' "cabe" em 'a' será o quociente e o valor restante da última subtraçăo será o resto. Partindo deste princípio, tomamos o módulo de 'a' e 'd' e aplicamos o algoritmo. Se ambos forem năo negativos já teremos a resposta. Para as demais situaçőes de sinal do dividendo e divisor tudo o que é preciso fazer é ajustar o quociente e o resto de forma a garantir que a soluçăo encontrada atenderá (1).

Algoritmo

// Preparaçăo das variáveis
 1. A = |a|, D = |d|, Q = 0, R = 0, sa = falso, sd = falso;
 2. se a < 0 entăo sa = verdadeiro;
 3. se d < 0 entăo sd = verdadeiro;
// Algoritmo básico
 4. enquanto A > D faça
       A = A - D;
       Q = Q + 1;
    fim (enquanto);
    R = A;
// Dividendo e divisor positivos
 5. se (sa = falso) e (sd = falso) entăo
       q = Q;
       r = R;
    fim (se)
    senăo
// Somente dividendo é negativo
 6. se (sa = verdadeiro) e (sd = falso) entăo
       q = -(Q+1);
       r = D - R;
    fim (se)
    senăo
// Somente divisor é negativo
 7. se (sa = falso) e (sd = verdadeiro) entăo
       q = -Q;
       r = R;
    fim (se)
    senăo
// ambos, dividendo e divisor, săo negativos
 8. se (sa = verdadeiro) e (sd = verdadeiro) entăo
       q = Q + 1;
       r = D - R;
    fim (se);
 9. FIM
